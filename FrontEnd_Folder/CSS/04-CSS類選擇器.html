韓國瑜圖樣水溝蓋？ 民進黨里長打造的

高市新興區德生里最近出現印有龍鳳、高市長韓國瑜牽著狗的圖樣的金色水溝蓋，為民進黨籍里長吳三雄催生打造。韓國瑜說，不管各種符號、圖案、設計，如高雄人覺得高雄發大財很開心，願意在各方面表達，他都樂觀其成。

對於金色水溝蓋，里長吳三雄表示，希望韓看到能認同，多採用他設計好用的水溝蓋。對有網友批他拍馬屁，他強調不可能，因自己是民進黨員、高市黨部執委，只是希望市長看了能多推廣使用。

韓國瑜上午在議會受訪表示，該名里長是民進黨籍，也講得很清楚，「不管各種符號、圖案、設計，如果高雄人大家都覺得高雄發大財，覺得非常開心，也願意在各種方面表達，包括小物等，我是樂觀其成，且他跟政治上支持我一點關係都沒有，因為他是民進黨籍，他也講得非常清楚。」
